# TiebaSign

[![npm package](https://img.shields.io/npm/v/tieba-sign.svg?style=flat-square)](https://www.npmjs.org/package/tieba-sign)

> sign tieba in command line

## Installation

```bash
$ npm i tieba-sign -g
```

## Usage

### Basic

```bash
# set cookie
$ ts cookie <bduss>
# sign
$ ts
```

### All Commands

Command | Description
------- | -----------
ts cookie | save cookie
ts | sign
